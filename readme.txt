== Description ==
Add custom contact forms in your wordpress website easily with this free contact form plugin. Customizable and re-captcha available.

Choose **which fields** are required and **the script will generate** a shortcode for you. 

Simpy add the **shortcode** to the **post** or **page** where you want to display the form.

= Contact Form - Features =
1.Customizable
2.Re-captcha
3.Unique Shortcode(s)

== Installation ==
1. Download and upload the file `contact-form.zip` from the plugin panel or unzip the file and upload the folder `contact-form` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` panel in your WordPress Site.
3. Go to `Settings-> Contact Form` to build your form and generate the shortcode.
4. Paste the shortcode where you want to display the form.
